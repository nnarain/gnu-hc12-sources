# gnu-hc12-sources
Sources for GNU HC12 Compiler

Requirements
------------

* build-essential
* lex
* bison

```bash
~$ sudo apt-get install build-essential flex bison
```

Build
-----

```bash
~$ cd path/to/gnu-hc12-sources
~$ ./mk-toolchain.sh
```

Download link to source tar balls
---------------------------------

[FreeHC12 Source tar balls](http://freehc12.gibibit.com/Download)
